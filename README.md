## Как использовать репозиторий
1. Все документы проекта хранятся в соответствующих папках.
2. Названия файлов должны отражать их содержание(например, `TZ_vnedrenia.doks`).
3. При изменении документов создайте новую версию, не удаляя старые (для сохранения истории).
